## 数据类型概念
- “类型”是对数据的抽象
- 类型相同的数据有相同的表示形式、存储格式以及相关的操作
- 程序中使用的所有数据都必定属于某一种数据类型

# 数据类型的本质
1. 数据类型可理解为创建变量的模具（模子）；
2. 是固定内存大小的别名。


## 数据类型的作用
- 编译器预算对象（变量）分配的内存空间大小

```c

#include<stdlib.h>
/*
数据类型的本质：

*/
void main()

		b+1 &b+1结果不一样 b  &b所代表的类据类型不一样

		&b 代表的是整个数组的地址

		b &b 数组数据类型：
		1.定义一个数组类型
		2 数组指针
		3 数组类型和数组指针类型的关系
	*/	
	//求长度
```
